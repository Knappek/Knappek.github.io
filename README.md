# Andy's Technical Scratchpad

I use this page at https://knappek.github.io/ to document all kinds of stuff because I am getting old and rusty and forget things. So this page is mainly for me but I'm happy if others can benefit from it as well 🙂.

## Running locally

1. Create a python environment and source it:

    ```sh
    python -m venv venv
    source venv/bin/activate
    ```

1. install all requirements

    ```sh
    pip install -r requirements.txt
    ```

1. run the page locally

    ```sh
    mkdocs serve
    ```
